# SQLite-рЃЊрЃљрЃю PostgreSQL-рЃќрЃћ рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ Supabase-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ

## рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў

рЃћрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ рЃњрЃљрЃњрЃљрЃбрЃљрЃарЃћрЃЉрЃЌ SQLite рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃЊрЃљрЃю Supabase-рЃўрЃА PostgreSQL рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃќрЃћ рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃА. Supabase рЃљрЃарЃўрЃА рЃдрЃўрЃљ рЃгрЃДрЃљрЃарЃЮрЃА Firebase рЃљрЃџрЃбрЃћрЃарЃюрЃљрЃбрЃўрЃЋрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА PostgreSQL-рЃА рЃЊрЃљ рЃњрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃЌ рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљрЃА, рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃћрЃЉрЃА рЃЊрЃљ API-рЃА.

## рЃюрЃљрЃЉрЃўрЃ»рЃў 1: Supabase рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

1. рЃерЃћрЃЊрЃўрЃЌ [Supabase](https://supabase.com/) рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў (рЃЌрЃБ рЃљрЃа рЃњрЃљрЃЦрЃЋрЃЌ)
2. рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў:
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "New Project" рЃдрЃўрЃџрЃљрЃЎрЃА
   - рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў
   - рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃарЃћрЃњрЃўрЃЮрЃюрЃў (рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃўрЃљ рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃљрЃю рЃљрЃ«рЃџрЃЮрЃА)
   - рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ рЃърЃљрЃарЃЮрЃџрЃў (рЃЊрЃљрЃўрЃЏрЃљрЃ«рЃАрЃЮрЃЋрЃарЃћрЃЌ рЃћрЃА рЃърЃљрЃарЃЮрЃџрЃў!)
   - рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "Create new project"

## рЃюрЃљрЃЉрЃўрЃ»рЃў 2: рЃАрЃЦрЃћрЃЏрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ

### 2.1 SQLite рЃАрЃЦрЃћрЃЏрЃўрЃА рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў

рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў, рЃБрЃюрЃЊрЃљ рЃЏрЃЮрЃЋрЃўрЃърЃЮрЃЋрЃЮрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃАрЃЦрЃћрЃЏрЃљ SQLite-рЃЊрЃљрЃю:

```bash
sqlite3 dev.db .schema > schema.sql
```

### 2.2 рЃАрЃЦрЃћрЃЏрЃўрЃА рЃљрЃЊрЃљрЃърЃбрЃљрЃфрЃўрЃљ PostgreSQL-рЃАрЃЌрЃЋрЃўрЃА

SQLite рЃЊрЃљ PostgreSQL рЃАрЃўрЃюрЃбрЃљрЃЦрЃАрЃў рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃЊрЃћрЃЉрЃљ. рЃБрЃюрЃЊрЃљ рЃерЃћрЃЋрЃфрЃЋрЃљрЃџрЃЮрЃЌ рЃАрЃЦрЃћрЃЏрЃљ PostgreSQL-рЃАрЃЌрЃЋрЃўрЃА:

1. рЃерЃћрЃфрЃЋрЃљрЃџрЃћрЃЌ рЃбрЃўрЃърЃћрЃЉрЃў:
   - `INTEGER PRIMARY KEY AUTOINCREMENT` Рєњ `serial PRIMARY KEY`
   - `TIMESTAMP DEFAULT CURRENT_TIMESTAMP` Рєњ `timestamp DEFAULT now()`
   - `BOOLEAN` рЃбрЃўрЃърЃћрЃЉрЃў рЃБрЃфрЃЋрЃџрЃћрЃџрЃљрЃЊ рЃарЃЕрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃўрЃљ рЃЊрЃљрЃўрЃгрЃћрЃарЃЮрЃА рЃърЃљрЃбрЃљрЃарЃљ рЃљрЃАрЃЮрЃћрЃЉрЃўрЃЌ: `boolean`

2. рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃБрЃЎрЃЋрЃћ рЃерЃћрЃўрЃфрЃљрЃЋрЃА PostgreSQL-рЃЌрЃЋрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃАрЃЦрЃћрЃЏрЃљрЃА. рЃўрЃ«рЃўрЃџрЃћрЃЌ `/Users/imac/test2/migrations/0000_ambitious_mephistopheles.sql` рЃЊрЃљ `/Users/imac/test2/migrations/0001_tips_tutorials.sql`.

### 2.3 рЃАрЃЦрЃћрЃЏрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ Supabase-рЃерЃў

рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃњрЃќрЃљ рЃАрЃЦрЃћрЃЏрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ Supabase-рЃерЃў:

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: SQL рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

1. Supabase рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃерЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ "SQL Editor" рЃњрЃљрЃюрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃљрЃерЃў
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "New Query"
3. рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў SQL рЃАрЃЦрЃћрЃЏрЃљ
4. рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљ

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2: Supabase CLI-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

1. рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЌ Supabase CLI:

```bash
npm install -g supabase
```

2. рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ:

```bash
supabase login
supabase init
```

3. рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў:

```bash
supabase link --project-ref YOUR_PROJECT_ID
```

4. рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ:

```bash
supabase migration new initial_schema
# рЃЕрЃљрЃАрЃЋрЃўрЃЌ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў SQL рЃАрЃЦрЃћрЃЏрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџ рЃцрЃљрЃўрЃџрЃерЃў
supabase migration up
```

## рЃюрЃљрЃЉрЃўрЃ»рЃў 3: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ

### 3.1 рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў SQLite-рЃЊрЃљрЃю

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃўрЃАрЃЌрЃЋрЃўрЃА SQLite-рЃЊрЃљрЃю CSV рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў:

```bash
# рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ
sqlite3 dev.db ".tables"

# рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃфрЃ«рЃарЃўрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА:
sqlite3 -header -csv dev.db "SELECT * FROM recipes;" > recipes.csv
sqlite3 -header -csv dev.db "SELECT * FROM ingredients;" > ingredients.csv
sqlite3 -header -csv dev.db "SELECT * FROM steps;" > steps.csv
sqlite3 -header -csv dev.db "SELECT * FROM news;" > news.csv
sqlite3 -header -csv dev.db "SELECT * FROM tutorials;" > tutorials.csv
sqlite3 -header -csv dev.db "SELECT * FROM tips;" > tips.csv
```

### 3.2 рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃў Supabase-рЃерЃў

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 1: Supabase рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

1. Supabase рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃерЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ "Table Editor" рЃњрЃљрЃюрЃДрЃЮрЃцрЃўрЃџрЃћрЃЉрЃљрЃерЃў
2. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃфрЃ«рЃарЃўрЃџрЃў
3. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "Import" рЃдрЃўрЃџрЃљрЃЎрЃА
4. рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ CSV рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "Import"
5. рЃњрЃљрЃўрЃЏрЃћрЃЮрЃарЃћрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃфрЃ«рЃарЃўрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 2: pgAdmin рЃљрЃю рЃАрЃ«рЃЋрЃљ PostgreSQL рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ

1. рЃЊрЃљрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЌ pgAdmin Supabase рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃАрЃЌрЃљрЃю (рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў рЃўрЃ«рЃўрЃџрЃћрЃЌ Supabase рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃерЃў "Settings" > "Database")
2. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ "Import/Export" рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ CSV рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃАрЃЌрЃЋрЃўрЃА

#### рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў 3: рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃўрЃЏрЃърЃЮрЃарЃбрЃў

рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃерЃћрЃЦрЃЏрЃюрЃљрЃЌ рЃАрЃЎрЃарЃўрЃърЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃљрЃЋрЃА CSV рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА Supabase-рЃерЃў API-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ:

```javascript
import { createClient } from '@supabase/supabase-js'
import fs from 'fs'
import csv from 'csv-parser'

const supabaseUrl = 'YOUR_SUPABASE_URL'
const supabaseKey = 'YOUR_SUPABASE_KEY'
const supabase = createClient(supabaseUrl, supabaseKey)

// рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў: recipes рЃфрЃ«рЃарЃўрЃџрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃў
fs.createReadStream('recipes.csv')
  .pipe(csv())
  .on('data', async (row) => {
    const { error } = await supabase
      .from('recipes')
      .insert([
        {
          title: row.title,
          description: row.description,
          image_url: row.image_url,
          category: row.category,
          prep_time: parseInt(row.prep_time),
          cook_time: parseInt(row.cook_time),
          servings: parseInt(row.servings),
          difficulty: row.difficulty,
          created_at: row.created_at,
          updated_at: row.updated_at
        }
      ])
    
    if (error) console.error('Error inserting recipe:', error)
  })
```

### 4.1 Supabase рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

### 4.4 рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃљрЃЊрЃљрЃърЃбрЃћрЃарЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ

рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ `server/db.ts` рЃцрЃљрЃўрЃџрЃў, рЃарЃЮрЃЏ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ PostgreSQL Drizzle рЃљрЃЊрЃљрЃърЃбрЃћрЃарЃў SQLite-рЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ:

```typescript
import { drizzle } from 'drizzle-orm/postgres-js'
import postgres from 'postgres'
import { recipes as georgianRecipes, ingredients as recipeIngredients, steps as recipeSteps, news, tutorials, tips } from '@shared/schema'
import { eq } from 'drizzle-orm'
import 'dotenv/config'

// рЃЉрЃљрЃќрЃўрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃАрЃбрЃарЃўрЃюрЃњрЃў
const connectionString = process.env.DATABASE_URL || ''

// PostgreSQL рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
const client = postgres(connectionString)
export const db = drizzle(client)

console.log(`­Ъћї Using PostgreSQL database`)

// рЃарЃћрЃфрЃћрЃърЃбрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ ID-рЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ
export async function getGeorgianRecipeById(id: number) {
  try {
    // рЃарЃћрЃфрЃћрЃърЃбрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
    const [recipe] = await db.select().from(georgianRecipes).where(
      eq(georgianRecipes.id, id)
    )

    if (!recipe) {
      return null
    }

    // рЃўрЃюрЃњрЃарЃћрЃЊрЃўрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
    const ingredients = await db.select().from(recipeIngredients).where(
      eq(recipeIngredients.recipeId, id)
    )

    // рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ
    const steps = await db.select().from(recipeSteps).where(
      eq(recipeSteps.recipeId, id)
    ).orderBy(recipeSteps.stepNumber)

    // рЃАрЃарЃБрЃџрЃў рЃарЃћрЃфрЃћрЃърЃбрЃўрЃА рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљ
    return {
      ...recipe,
      ingredients,
      steps
    }
  } catch (error) {
    console.error('Error getting recipe by ID:', error)
    throw error
  }
}

// рЃЊрЃљрЃюрЃљрЃарЃЕрЃћрЃюрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў...
```

## рЃюрЃљрЃЉрЃўрЃ»рЃў 5: рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

2. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЎрЃЮрЃарЃћрЃЦрЃбрЃБрЃџрЃљрЃЊ

## рЃюрЃљрЃЉрЃўрЃ»рЃў 6: рЃЊрЃћрЃърЃџрЃЮрЃўрЃЏрЃћрЃюрЃбрЃў

1. рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ Render.com-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃљрЃ«рЃљрЃџрЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃЌ:

2. рЃњрЃљрЃЊрЃљрЃљрЃњрЃќрЃљрЃЋрЃюрЃћрЃЌ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў GitHub-рЃќрЃћ рЃЊрЃљ рЃЊрЃћрЃърЃџрЃЮрЃў рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЌ Render.com-рЃќрЃћ

## рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў

1. **рЃЉрЃћрЃЦрЃљрЃърЃў**: рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ SQLite рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃЉрЃћрЃЦрЃљрЃърЃў рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљрЃЏрЃЊрЃћ
2. **рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ**: рЃ»рЃћрЃа рЃбрЃћрЃАрЃбрЃБрЃа рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў рЃњрЃљрЃбрЃћрЃАрЃбрЃћрЃЌ рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ
3. **рЃўрЃюрЃЊрЃћрЃЦрЃАрЃћрЃЉрЃў**: рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃўрЃюрЃЊрЃћрЃЦрЃАрЃў рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃџрЃўрЃљ PostgreSQL-рЃерЃў
4. **рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃў**: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃћрЃЉрЃў рЃЊрЃўрЃЊрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃЮрЃарЃбрЃўрЃАрЃљрЃА
5. **рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў**: рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ Supabase-рЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃЊрЃњрЃћрЃюрЃљрЃЊ

## Supabase-рЃўрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў

- **рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃарЃћрЃЉрЃљ**: PostgreSQL рЃБрЃЎрЃћрЃЌрЃћрЃАрЃљрЃЊ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃарЃЊрЃћрЃЉрЃљ, рЃЋрЃўрЃЊрЃарЃћ SQLite
- **рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў**: рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ
- **рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў**: рЃљрЃЋрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃАрЃбрЃЮрЃарЃўрЃ»рЃў, рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃћрЃЉрЃў
- **API**: рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў RESTful рЃЊрЃљ GraphQL API
- **рЃћрЃЦрЃАрЃбрЃћрЃюрЃфрЃўрЃћрЃЉрЃў**: PostgreSQL-рЃўрЃА рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃћрЃЦрЃАрЃбрЃћрЃюрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ

рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃўрЃАрЃљрЃарЃњрЃћрЃЉрЃџрЃћрЃЉрЃА Supabase-рЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃўрЃЌ рЃЊрЃљ рЃЏрЃќрЃљрЃЊ рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЏрЃљрЃАрЃерЃбрЃљрЃЉрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
